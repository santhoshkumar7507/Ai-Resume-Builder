# 🚀 AI Resume Builder & Analyzer

[![Puter.js](https://img.shields.io/badge/Powered%20by-Puter.js-blueviolet?style=for-the-badge)](https://puter.com)
[![React](https://img.shields.io/badge/React-19-61DAFB?style=for-the-badge&logo=react)](https://react.dev)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-4.0-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com)

**AI Resume Builder** is a premium, high-performance web application designed to help job seekers optimize their resumes using cutting-edge AI. Built with **Puter.js**, **React**, and **Tailwind CSS**, it provides real-time ATS scoring, professional feedback, and a seamless user experience.

---

## ✨ Key Features

- **🧠 AI-Powered Analysis**: Get instant feedback on your resume based on specific job descriptions.
- **🎯 ATS Scoring**: Understand how your resume ranks against industry-standard Applicant Tracking Systems.
- **🖼️ Real-time Preview**: Upload your PDF and see an instant visual preview of your resume.
- **🔐 Secure & Private**: All data is stored and processed securely via **Puter.js** cloud storage.
- **🎨 Premium UI/UX**: A stunning, modern interface with smooth animations, dark mode support, and glassmorphism effects.
- **📁 Dashboard**: Manage all your analyzed resumes in one beautiful, organized view.

---

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://reactjs.org/), [React Router 7](https://reactrouter.com/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/), [Lucide Icons](https://lucide.dev/)
- **Cloud Infrastructure**: [Puter.js](https://puter.com) (Auth, KV Storage, File System, AI)
- **PDF Processing**: [PDF.js](https://mozilla.github.io/pdf.js/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/santhoshkumar7507/Ai-Resume-Builder.git
   cd Ai-Resume-Builder
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
